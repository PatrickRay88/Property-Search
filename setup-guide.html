Required API Keys

1. OpenAI API Key (Required for AI Features)

Step 1: Get Your API Key
  1. Go to OpenAI Platform
  2. Sign up or log in to your account
  3. Navigate to "API Keys" in the left sidebar
  4. Click "Create new secret key"
  5. Copy the key (starts with sk-...)
  6. Important: Add billing information (required even for free tier)
Step 2: Add Key to Configuration
  1. Open ai-config.js in your project folder
  2. Replace the placeholder with your actual key:
OPENAI_API_KEY: 'sk-your-actual-key-here',
  1. Save the file
Step 3: Test the Setup
  1. Refresh your browser (http://localhost:3000)
  2. The orange setup banner should disappear
  3. Try the AI Natural Language Search
  4. Check browser console (F12) for confirmation messages
